# my portfolion website 

This is my portfolio using js and typeScript and tailwind 
## if you wnna use just clone the repo :
clone the project 
```bash
$ git clone git@github.com:chahrazad-sofia/simple-tailwind-react.git
```
got to the directory 
```bash
$ cd simple-tailwind-react
```
install dependencies 
```bash
$ npm install
#or
$ yarn install
```
run the project
```bash
$ npm run dev 
#or
$ yarn dev
```
